# Sample Reports

This folder contains example reports demonstrating the output of the Azure Savings Realization Report Generator.

## Files

| File | Description |
|------|-------------|
| [sample_report_contoso_20251204.html](sample_report_contoso_20251204.html) | Example HTML report with interactive charts |
| [sample_report_contoso_20251204.xlsx](sample_report_contoso_20251204.xlsx) | Example Excel report with detailed data |
| [generate_sample_excel.py](generate_sample_excel.py) | Script to regenerate the sample Excel file |

## Sample Data Overview

The example reports show data for a fictional customer "Contoso Corporation" with:

- **Report Period**: September 1, 2025 - December 1, 2025 (3 months)
- **Total Retail Cost**: $2,714,850
- **Total Effective Cost**: $1,867,616
- **Total Savings**: $847,234 (31.2%)

### Savings Breakdown

| Category | Savings | Description |
|----------|---------|-------------|
| Negotiated Discount | $271,485 | 10% EA contract discount vs retail |
| Reserved Instances | $312,450 | 62% average RI discount |
| Savings Plans | $156,780 | 38% compute savings plan discount |
| Azure Hybrid Benefit | $78,519 | Windows & SQL Server license savings |
| Dev/Test Pricing | $28,000 | Dev/Test subscription discounts |

## Viewing the HTML Report

Open `sample_report_contoso_20251204.html` in any web browser to see the interactive report with:

- Executive summary cards
- Savings distribution pie chart
- Monthly trend bar/line chart
- Detailed tables for services, subscriptions, and resources

## Excel Report Sheets

The Excel file contains the following worksheets:

1. **Executive Summary** - High-level metrics and totals
2. **By Service** - Savings breakdown by Azure service category
3. **By Subscription** - Savings breakdown by subscription
4. **Monthly Trend** - Month-over-month cost and savings data
5. **Top Resources** - Resources with highest savings
6. **Reserved Instances** - RI details and utilization
7. **Savings Plans** - Savings plan details and utilization
8. **Azure Hybrid Benefit** - AHB usage by license type

## Regenerating Sample Files

To regenerate the Excel sample:

```powershell
cd examples
python generate_sample_excel.py
```

The HTML file is static and can be edited directly or regenerated by running the full report generator with sample data.
